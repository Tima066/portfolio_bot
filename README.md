# Telegram-Бот Портфолио | Тамирлан

Бот-портфолио "Обо мне", созданный в рамках конкурса портфолио курса Python. Бот содержит 8 обязательных контентных блоков и демонстрирует навыки работы с библиотекой `telebot`, регулярными выражениями и аргументами командной строки.

## Реализованные проекты внутри портфолио
1. **Игра «Дино» на Pygame** аркада со звуковым сопровождением, бот так же показывает видео с геймплеем.
2. **Программа «Обучающие карточки»** приложение для запоминания желаемой информации, бот показывает фото альбом работы кода.
3. **Telegram-бот «Water Reminder»** бот для отслеживания дневной нормы воды, бот показывает фото с его функционалом.

## Технические особенности проекта
- **Разделение логики и контента**: все тексты вынесены в отдельный файл `content.json`, что позволяет менять информацию без вмешательства в главный код программы.
- **Аргументы командной строки**: реализована безопасная передача токена бота через список `sys.argv` (модуль `sys`).
- **Регулярные выражения (Regex)**: использование модуля `re` в `utils.py` для валидации почты пользователя при отправке контактов.
- **Стабильность**: все кнопки защищены блоками `try-except` для предотвращения ошибок в работе бота и полной его остановки при отсутствии медиафайлов.

## Инструкция по запуску

1. скопируйте репозиторий и перейдите в папку проекта.
2. установите необходимые сторонние зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. запустите бота, передав токен от @BotFather в качестве аргумента командной строки:
   ```bash
   python main.py
   ```
   токен от @BotFather должен быть после команды `python main.py` и выглядеть вот так:
   ```bash
   python main.py "ваш токен от @BotFather"
   ```
   если при запуске серез консоль вы не вписываете токен для бота то код запускает дефолтного бота со своим токеном которого вы найдете по тегу
   ```bash
   @Tamirlan_portfolio_bot
   ```
